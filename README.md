##   What is it?

Swift Extension to convert CMYK values into UIColor Objects.



## Installation

#### Podfile

```ruby
platform :ios, '8.0'
use_frameworks!

pod 'ColorWithCMYK', '~> 0.0'
```

Then, run the following command:

```bash
$ pod install
```

#### Old school

Drag into your project the folder `/ColorWithCMYK-iOS`. That's all.
